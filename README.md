## 💻 Manual Installation (VPS / Panel / Terminal)

Follow these steps to set up the bot on your local machine or a Linux server.

### 📥 Step 1: Clone the Repository
Open your terminal and run the following command to download the project files.

```bash
git clone https://github.com/arun-cumar/Nexa-Bot-Md.git && cd Nexa-Bot-Md
```

🏗️ Step 2: Install Dependencies
Install the required Node.js packages to ensure the bot functions correctly.
```bash
npm install
```
🚢 Step 3: Configure and Start
Once the installation is complete, you can launch the bot using the command below. Make sure you have your SESSION_ID ready.
# If you have a config.js file, edit it first, then run:
```bash
npm start
```
⚙️ Configuration
Before running npm start, ensure you have your environment variables set up (either in a .env file or directly in config.js):
 * SESSION_ID: Your json session code.
 * PREFIX: The symbol used to trigger commands (e.g., . or !).
 * OWNER_NUMBER: Your WhatsApp number.
